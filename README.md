- [vite-plugin-gleam](https://github.com/gleam-br/vite-plugin-gleam)

## How to

Help me:

```sh
bun create vite-lustre --help # or -h
```

Create `my-first-lustre`:

```sh
bun create vite-lustre my-first-lustre
cd my-first-lustre
```

Dev `my-first-lustre`:

```sh
bun install
bun run dev
```

Build `my-first-lustre`:

```sh
bun install
bun run build
```
